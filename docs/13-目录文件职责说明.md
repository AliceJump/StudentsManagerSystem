## `StudentsManagerSystem/Common/`

### `ObservableObject.cs`

- 学年显示转换器。

## `StudentsManagerSystem/Data/`

### `StudentsManagerDbContext.cs`
- EF Core 数据库上下文、实体映射和表结构配置。

### `StudentsManagerDbContextFactory.cs`
- 创建数据库上下文实例。

### `DatabaseConfiguration.cs`
- 数据库连接字符串和配置读取。

### `DatabaseInitializer.cs`
- 数据库初始化、建表和种子数据写入。

#### `Data/Repositories/`
- 仓储层目录，封装 SQLite + EF Core 的数据访问细节。

##### `StudentRepository.cs`
- 学生基本信息和学生档案子表的数据访问。

##### `StudentStatusRepository.cs`
- 学籍注册、学籍变动、奖助学金和毕业信息的数据访问。

##### `ScoreRepository.cs`
- 成绩、课程和成绩相关查询的数据访问。

##### `BasicDataRepository.cs`
- 院系、专业、班级和通用字典的数据访问。

##### `UsersRepository.cs`
- 用户登录验证与用户数据访问。

#### `Data/Migrations/`
- EF Core 迁移文件目录，用于数据库结构演进。

## 补充说明

- 目录说明以当前代码结构为准，新增模块时应同步更新本文档。
- 运行时逻辑应优先沿用 `View -> Service -> Repository -> DbContext` 的调用链。
