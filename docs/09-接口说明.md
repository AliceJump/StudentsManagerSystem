# 09-接口说明

## 说明

本项目不是 Web API 或 Service 项目，没有 HTTP 接口、Controller 路由或外部 RPC 接口。本文档中的“接口”指应用内部对外调用入口：Service 方法、Repository 方法、UI 事件入口和配置项。

## Service 接口

### StudentService

- [`GetAll()`](../StudentsManagerSystem/Services/StudentService.cs#L13)：获取全部学生。
- [`Search(string keyword)`](../StudentsManagerSystem/Services/StudentService.cs#L15)：按学号、姓名、院系、专业、班级搜索学生。
- [`GetFamilyInfos()`](../StudentsManagerSystem/Services/StudentService.cs#L17)、[`GetRewardRecords()`](../StudentsManagerSystem/Services/StudentService.cs#L19)、[`GetPunishmentRecords()`](../StudentsManagerSystem/Services/StudentService.cs#L21)、[`GetHealthRecords()`](../StudentsManagerSystem/Services/StudentService.cs#L23)：读取学生档案子表。
- [`SaveFamilyInfo(FamilyInfo)`](../StudentsManagerSystem/Services/StudentService.cs#L27)、[`SaveRewardRecord(RewardRecord)`](../StudentsManagerSystem/Services/StudentService.cs#L62)、[`SavePunishmentRecord(PunishmentRecord)`](../StudentsManagerSystem/Services/StudentService.cs#L84)、[`SaveHealthRecord(HealthRecord)`](../StudentsManagerSystem/Services/StudentService.cs#L105)：保存学生档案子表记录，新增和修改共用。
- [`DeleteFamilyInfo(int id)`](../StudentsManagerSystem/Data/Repositories/StudentRepository.cs#L136)、[`DeleteRewardRecord(int id)`](../StudentsManagerSystem/Data/Repositories/StudentRepository.cs#L156)、[`DeletePunishmentRecord(int id)`](../StudentsManagerSystem/Data/Repositories/StudentRepository.cs#L171)、[`DeleteHealthRecord(int id)`](../StudentsManagerSystem/Data/Repositories/StudentRepository.cs#L186)：删除学生档案子表记录。
- [`Add(Student student)`](../StudentsManagerSystem/Data/Repositories/StudentRepository.cs#L201)：新增学生并校验。
- [`Update(Student student)`](../StudentsManagerSystem/Data/Repositories/StudentRepository.cs#L216)：更新学生并校验。
- [`SaveImported(Student student)`](../StudentsManagerSystem/Services/StudentService.cs)：导入时按学号新增或更新。
- [`Delete(int id)`](../StudentsManagerSystem/Data/Repositories/StudentRepository.cs#L248)：删除学生和关联数据。
- `Validate(Student student, int excludeId)`：学生格式和重复校验。

### ScoreService

- [`GetAll()`](../StudentsManagerSystem/Services/ScoreService.cs#L17)：获取全部成绩。
- [`GetByAcademicYearSemester(string, string)`](../StudentsManagerSystem/Services/ScoreService.cs#L19)：按学年学期查询。
- [`GetCourses()`](../StudentsManagerSystem/Services/ScoreService.cs#L21)：获取课程。
- [`GetAcademicYears()`](../StudentsManagerSystem/Services/ScoreService.cs#L23)：获取成绩中已有学年。
- [`GetSemesters()`](../StudentsManagerSystem/Services/ScoreService.cs#L25)：从字典读取学期。
- [`Add/Update/Delete`](../StudentsManagerSystem/Services/ScoreService.cs#L29)：成绩增删改。
- [`SaveImported(Score score)`](../StudentsManagerSystem/Services/ScoreService.cs#L67)：导入时按学号、学年、学期、课程编号合并。

### BasicDataService

- [`GetDepartments()`](../StudentsManagerSystem/Services/BasicDataService.cs#L12)、[`GetMajors()`](../StudentsManagerSystem/Services/BasicDataService.cs#L14)、[`GetClasses()`](../StudentsManagerSystem/Services/BasicDataService.cs#L16)：获取基础数据。
- [`GetDepartmentNames()`](../StudentsManagerSystem/Services/BasicDataService.cs#L18)、[`GetMajorNames()`](../StudentsManagerSystem/Services/BasicDataService.cs#L20)、[`GetClassNames()`](../StudentsManagerSystem/Services/BasicDataService.cs#L22)：获取下拉显示名称。
- [`GetLookupValues(string category)`](../StudentsManagerSystem/Services/BasicDataService.cs#L26)：获取通用字典。
- [`Add/Update/Delete Department/Major/Class`](../StudentsManagerSystem/Services/BasicDataService.cs#L29)：基础数据增删改。

## Repository 接口

Repository 只面向内部 Service 或 View 调用，不作为外部接口。

- [`StudentRepository`](../StudentsManagerSystem/Data/Repositories/StudentRepository.cs#L10)：学生与档案子表数据访问。
- [`ScoreRepository`](../StudentsManagerSystem/Data/Repositories/ScoreRepository.cs#L10)：成绩、课程、学年、学期数据访问。
- [`BasicDataRepository`](../StudentsManagerSystem/Data/Repositories/BasicDataRepository.cs#L10)：院系、专业、班级、字典数据访问。
- [`StudentStatusRepository`](../StudentsManagerSystem/Data/Repositories/StudentStatusRepository.cs#L10)：学籍相关数据访问。
- [`UsersRepository`](../StudentsManagerSystem/Data/Repositories/UsersRepository.cs#L10)：登录验证。

## UI 事件入口

- [`StudentArchiveView.btnAdd_Click`](../StudentsManagerSystem/Views/StudentArchive/StudentArchiveView.xaml.cs#L13)、`btnEdit_Click`、`btnDelete_Click`、`btnView_Click`、`btnImport_Click`、`btnExport_Click`：学生档案操作。
- [`ScoreView.btnQuery_Click`](../StudentsManagerSystem/Views/Score/ScoreView.xaml.cs#L68)、[`btnImport_Click`](../StudentsManagerSystem/Views/Score/ScoreView.xaml.cs#L145)、`btnExport_Click`：成绩操作。
- [`BasicDataView.btnAdd_Click`](../StudentsManagerSystem/Views/BasicData/BasicDataView.xaml.cs#L10)、`btnEdit_Click`、`btnDelete_Click`：基础数据操作。
- [`QueryView.btnQuery_Click`](../StudentsManagerSystem/Views/Query/QueryView.xaml.cs#L104)、`btnReset_Click`、`btnExport_Click`：查询统计操作。

## 配置项

配置文件：[`StudentsManagerSystem/App.config`](../StudentsManagerSystem/App.config)

- `DatabaseFile`：SQLite 数据库文件名。
- `AdminUsername`：初始化管理员用户名。
- `AdminPassword`：初始化管理员密码。
- `StudentsManagerSystemDb`：SQLite 连接字符串。

## 外部依赖接口

- 文件系统：CSV 导入导出、日志写入。
- SQLite 数据库文件：默认 `StudentsManagerSystem.db`。
- WPF 文件对话框：`OpenFileDialog`、`SaveFileDialog`。
