# 06-调用链分析

## 应用启动链路

```text
App.OnStartup
-> AppLogger.Info
-> DatabaseInitializer.Initialize
-> StudentsManagerDbContextFactory.CreateDbContext
-> StudentsManagerDbContext.Database.EnsureCreated
-> SeedLookupOptions / SeedDepartments / SeedStudents ...
-> LoginWindow.ShowDialog
-> UsersRepository.ValidateCredentials
-> MainWindow.Show
```

返回路径：数据库初始化或登录失败时弹窗并中止；登录成功后显示主窗口。

## 学生新增链路

```text
StudentArchiveView.btnAdd_Click
-> StudentEditWindow.ShowDialog
-> StudentEditWindow.btnSave_Click
-> StudentService.Validate
-> StudentArchiveView receives ResultStudent
-> StudentService.Add
-> StudentRepository.Add
-> StudentsManagerDbContext.Students.Add
-> SQLite
```

返回路径：Repository 返回 `student.Id`，Service 返回 `ServiceResult`，View 根据结果刷新列表或显示错误。

## 学生搜索/排序/分页链路

```text
StudentArchiveView.btnSearch_Click
-> StudentService.Search
-> StudentRepository.Search
-> EF.Functions.Like query
-> SQLite
-> StudentArchiveView.ApplySortAndPaging
-> DataGrid.ItemsSource
```

## 成绩录入链路

```text
ScoreView.btnAdd_Click
-> ScoreEditWindow.ShowDialog
-> ScoreEditWindow.LoadCourses
-> ScoreService.GetCourses
-> ScoreRepository.GetCourses
-> ScoreEditWindow.TryBuildScore

```

## 成绩导入链路

```text
ScoreView.btnImport_Click
-> OpenFileDialog
-> ImportScores(file)
-> ScoreService.SaveImported
-> ScoreRepository.GetAll
-> ScoreService.Add or Update
-> ScoreRepository.Add or Update
-> AppLogger.Info
```

## 基础数据保存链路

```text
BasicDataView.btnAdd_Click / btnEdit_Click
-> BasicDataEditWindow.ShowDialog
-> BasicDataService.AddDepartment / UpdateDepartment 等
-> BasicDataRepository

```


## 权限控制链路

```text
LoginWindow
-> UsersRepository.ValidateCredentials(out role)
-> App.CurrentUserRole
-> MainWindow.ApplyPermissions
-> btnBasicData.IsEnabled = false for non-Admin
```

## 典型异常处理

- 启动初始化异常：`App.OnStartup` 捕获，写日志并弹窗后 `Shutdown(-1)`。
- UI 未处理异常：`DispatcherUnhandledException` 写日志并弹窗。
- 导入导出异常：页面 catch 后写日志并弹窗。
