# 01-项目概览

## 项目类型

本项目是 Desktop 桌面应用，不是 Web、Mobile、CLI、Library 或后台 Service。

- UI 技术：WPF
- 语言：C#
- 运行目标：Windows 桌面程序
- 项目文件：`StudentsManagerSystem/StudentsManagerSystem.csproj`
- 入口文件：`StudentsManagerSystem/App.xaml.cs`

## 项目定位

学生管理系统用于课程项目中的学生基础信息、学籍、成绩、基础数据、查询统计、登录权限和日志管理。当前已接入 SQLite + EF Core，并逐步从界面原型演进到真实业务数据接入。

## 当前能力概览

- 登录：启动后显示 `Views/Login/LoginWindow.xaml`，验证通过后进入主窗口。
- 主导航：`MainWindow.xaml` 左侧菜单加载学生档案、学籍、成绩、基础数据、查询统计页面。
- 学生档案：学生基本信息已支持 CRUD、搜索、排序、分页、CSV 导入导出；家庭、奖励、处分、体检已支持数据库 CRUD 和查看详情。
- 学籍管理：学籍注册、学籍变动、奖助学金、毕业信息已支持列表、搜索、新增、修改、删除和查看。
- 成绩管理：成绩列表、录入、修改、删除、详情、CSV 导入导出。
- 基础数据：院系、专业、班级 CRUD。
- 查询统计：学生条件查询、基础统计文本、CSV 导出。
- 日志：应用启动、异常、登录、导入导出、关键增删改写入日志。

## 核心技术栈

- .NET 8.0 Windows
- WPF XAML
- EF Core 8.0.8
- SQLite
- Microsoft.Data.SqlClient，当前仅遗留连接工厂引用，主要数据链路使用 SQLite
- System.Configuration.ConfigurationManager
