# 08-关键类说明

## 应用入口

### `App`

- 路径：`StudentsManagerSystem/App.xaml.cs`
- 作用：应用启动、全局异常处理、数据库初始化、登录流程、主窗口显示。
- 主要方法：`OnStartup`。
- 对外能力：提供 `CurrentUserRole`、`CurrentUserDisplayName` 静态属性。
- 依赖：`AppLogger`、`DatabaseInitializer`、`LoginWindow`、`MainWindow`。

### `MainWindow`

- 路径：`StudentsManagerSystem/MainWindow.xaml.cs`
- 作用：主界面导航、时间更新、基础权限控制。
- 主要方法：`ApplyPermissions`、`MenuButton_Click`。
- 依赖：`MainViewModel`、各功能页。

## 数据核心

### `StudentsManagerDbContext`

- 路径：`StudentsManagerSystem/Data/StudentsManagerDbContext.cs`
- 作用：EF Core 上下文和实体映射。
- 主要方法：`OnConfiguring`、`OnModelCreating`。
- 对外能力：提供所有业务实体的 `DbSet`。

### `DatabaseInitializer`

- 路径：`StudentsManagerSystem/Data/DatabaseInitializer.cs`
- 作用：创建数据库、创建表、补齐轻量结构、写入种子数据。
- 主要方法：`Initialize`、`SeedLookupOptions`、`SeedStudents`、`SeedScores`。
- 约束：唯一默认数据维护入口。

### `StudentsManagerDbContextFactory`

- 路径：`StudentsManagerSystem/Data/StudentsManagerDbContextFactory.cs`
- 作用：创建配置好 SQLite 连接的 DbContext。

## Service

### `StudentService`

- 路径：`StudentsManagerSystem/Services/StudentService.cs`
- 作用：学生查询、保存、导入合并、学生档案子表读取、校验、日志。
- 主要方法：`GetAll`、`Search`、`Add`、`Update`、`SaveImported`、`Validate`。
- 依赖：`StudentRepository`、`InputValidator`、`AppLogger`。

### `ScoreService`

- 路径：`StudentsManagerSystem/Services/ScoreService.cs`
- 作用：成绩查询、课程/学年/学期读取、成绩保存、导入合并、校验、日志。
- 主要方法：`GetByAcademicYearSemester`、`GetCourses`、`SaveImported`。
- 依赖：`ScoreRepository`。

### `BasicDataService`

- 路径：`StudentsManagerSystem/Services/BasicDataService.cs`
- 作用：基础数据 CRUD、字典读取、校验、日志。
- 主要方法：`GetDepartments`、`GetLookupValues`、`AddDepartment`、`UpdateClass`。
- 依赖：`BasicDataRepository`、`InputValidator`。

## Repository

### `StudentRepository`

- 路径：`StudentsManagerSystem/Data/Repositories/StudentRepository.cs`
- 作用：学生和学生档案子表数据访问。
- 主要方法：`GetAll`、`Search`、`Add`、`Update`、`Delete`、`GetFamilyInfos`。
- 调用方：`StudentService`。

### `StudentStatusRepository`

- 路径：`StudentsManagerSystem/Data/Repositories/StudentStatusRepository.cs`
- 作用：学籍、学籍变动、奖助、毕业数据访问。
- 调用方：`Views/StudentStatus/*Window.xaml.cs`、`StudentStatusView.xaml.cs`。

### `ScoreRepository`

- 路径：`StudentsManagerSystem/Data/Repositories/ScoreRepository.cs`
- 作用：成绩和课程数据访问。
- 调用方：`ScoreService`。

### `BasicDataRepository`

- 路径：`StudentsManagerSystem/Data/Repositories/BasicDataRepository.cs`
- 作用：院系、专业、班级、字典数据访问。
- 调用方：`BasicDataService`。

### `UsersRepository`

- 路径：`StudentsManagerSystem/Data/Repositories/UsersRepository.cs`
- 作用：登录凭据校验和最近登录时间更新。
- 调用方：`LoginWindow`。

## 通用类

- `InputValidator`：格式校验。
- `CsvExportHelper`：反射导出 CSV。
- `AppLogger`：文本日志。
- `AcademicYearHelper`：学年归一化。
- `AcademicYearDisplayConverter`：学年显示转换。
- `RelayCommand`、`AsyncRelayCommand`：WPF 命令。
- `ObservableObject`、`ViewModelBase`：属性变更通知。

## 源码文件职责清单

### 应用根文件

- `StudentsManagerSystem/App.xaml`：应用资源字典入口，合并 `Themes/ModernStyles.xaml`。
- `StudentsManagerSystem/App.xaml.cs`：应用启动、数据库初始化、登录流程、全局异常处理、当前用户角色保存。
- `StudentsManagerSystem/MainWindow.xaml`：主窗口布局、左侧导航、右侧 Frame。
- `StudentsManagerSystem/MainWindow.xaml.cs`：主窗口导航事件、时间刷新、基础数据入口权限控制。
- `StudentsManagerSystem/App.config`：数据库文件、管理员账号、SQLite 连接字符串配置。
- `StudentsManagerSystem/AssemblyInfo.cs`：WPF 主题程序集属性。
- `StudentsManagerSystem/StudentsManagerSystem.csproj`：项目目标框架、WPF 开关、NuGet 依赖。

### Common

- `Common/AcademicYearDisplayConverter.cs`：把学年起始年转换为显示文本。
- `Common/AcademicYearHelper.cs`：标准化学年输入。
- `Common/AppLogger.cs`：日志目录创建和日志写入。
- `Common/AsyncRelayCommand.cs`：异步 WPF 命令实现。
- `Common/CsvExportHelper.cs`：CSV 导出、值格式化和转义。
- `Common/InputValidator.cs`：学号、身份证、手机号、邮箱、姓名、学年、学期、金额校验。
- `Common/ObservableObject.cs`：属性变更通知基类。
- `Common/RelayCommand.cs`：同步 WPF 命令实现。

### Data

- `Data/DatabaseConfiguration.cs`：解析 SQLite 连接字符串和数据库文件路径。
- `Data/DatabaseInitializer.cs`：统一数据库初始化、结构补齐、种子数据维护。
- `Data/StudentsManagerDbContext.cs`：EF Core DbContext 和实体映射。
- `Data/StudentsManagerDbContextFactory.cs`：创建 DbContext 实例。
- `Data/Repositories/BasicDataRepository.cs`：院系、专业、班级、字典数据访问。
- `Data/Repositories/ScoreRepository.cs`：成绩、课程、学年、学期数据访问。
- `Data/Repositories/StudentRepository.cs`：学生、家庭、奖励、处分、体检数据访问。
- `Data/Repositories/StudentStatusRepository.cs`：学籍注册、变动、奖助、毕业数据访问。
- `Data/Repositories/UsersRepository.cs`：用户登录校验和最近登录时间更新。
- `Data/Migrations/*`：EF Core 数据库结构迁移。

### Models

- `Models/BasicData.cs`：`Department`、`Major`、`Class`、`LookupOption` 实体。
- `Models/Score.cs`：`Score`、`Course` 实体。
- `Models/Student.cs`：`Student`、`FamilyInfo`、`RewardRecord`、`PunishmentRecord`、`HealthRecord` 实体。
- `Models/StudentStatus.cs`：`StudentRegistration`、`StatusChangeRecord`、`ScholarshipInfo`、`GraduationInfo` 实体。
- `Models/User.cs`：登录用户实体。

### Services

- `Services/BasicDataService.cs`：基础数据业务校验、CRUD、字典读取、日志。
- `Services/ScoreService.cs`：成绩业务校验、课程/学年/学期读取、导入合并、日志。
- `Services/ServiceResult.cs`：Service 层统一成功/失败结果。
- `Services/StudentService.cs`：学生业务校验、CRUD、档案子表读取、导入合并、日志。

### ViewModels

- `ViewModels/MainViewModel.cs`：主窗口标题、时间、菜单项和命令。
- `ViewModels/StudentArchiveViewModel.cs`：学生档案 ViewModel 骨架，当前未被页面完整使用。
- `ViewModels/ViewModelBase.cs`：ViewModel 基类。

### Views/Login

- `Views/Login/LoginWindow.xaml`：登录窗口 UI。
- `Views/Login/LoginWindow.xaml.cs`：登录按钮、取消按钮、调用 `UsersRepository`。

### Views/StudentArchive

- `Views/StudentArchive/StudentArchiveView.xaml`：学生档案页面、Tab、搜索、排序、分页、导入导出按钮和 DataGrid。
- `Views/StudentArchive/StudentArchiveView.xaml.cs`：学生列表加载、搜索分页、子表读取、CSV 导入导出、增删改入口。
- `Views/StudentArchive/StudentEditWindow.xaml`：学生编辑表单。
- `Views/StudentArchive/StudentEditWindow.xaml.cs`：学生表单赋值、业务选项绑定、保存校验。
- `Views/StudentArchive/StudentDetailWindow.xaml`：学生详情窗口布局。
- `Views/StudentArchive/StudentDetailWindow.xaml.cs`：学生详情数据展示。

### Views/StudentStatus

- `Views/StudentStatus/StudentStatusView.xaml`：学籍管理主页面和 Tab。
- `Views/StudentStatus/StudentStatusView.xaml.cs`：学籍四类数据加载、搜索、增删改查入口。
- `Views/StudentStatus/RegistrationEditWindow.xaml`：学籍注册编辑表单。
- `Views/StudentStatus/RegistrationEditWindow.xaml.cs`：注册记录编辑、校验、保存。
- `Views/StudentStatus/StatusChangeEditWindow.xaml`：学籍变动编辑表单。
- `Views/StudentStatus/StatusChangeEditWindow.xaml.cs`：变动记录编辑、审批状态字典绑定、保存。
- `Views/StudentStatus/ScholarshipEditWindow.xaml`：奖助学金编辑表单。

- `Views/Score/ScoreView.xaml`：成绩列表、筛选、导入导出 UI。

- `Views/BasicData/BasicDataView.xaml`：基础数据页面。
- `Views/BasicData/BasicDataView.xaml.cs`：院系、专业、班级列表和增删改入口。

### Views/Query

- `Views/Query/QueryView.xaml`：查询条件、统计摘要、结果表格。
