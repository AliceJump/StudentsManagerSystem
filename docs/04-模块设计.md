# 04-模块设计

## 用户登录与权限模块

- 功能范围：登录校验、角色记录、基础数据入口权限。
- 涉及文件：`Views/Login/LoginWindow.xaml.cs`、`Data/Repositories/UsersRepository.cs`、`Models/User.cs`、`App.xaml.cs`、`MainWindow.xaml.cs`。
- 数据库表：`Users`。
- 依赖关系：LoginWindow -> UsersRepository -> DbContext；MainWindow 读取 `App.CurrentUserRole`。

## 主界面导航模块

- 功能范围：左侧导航、页面加载、系统时间、基础权限入口禁用。
- 涉及文件：`MainWindow.xaml`、`MainWindow.xaml.cs`、`ViewModels/MainViewModel.cs`。
- 数据库表：无直接表依赖。
- 依赖关系：MainWindow 加载各 Views。

## 学生档案模块

- 功能范围：学生基本信息 CRUD、搜索、排序、分页、CSV 导入导出；家庭、奖励、处分、体检 CRUD 和详情查看。
- 涉及文件：`Views/StudentArchive/*`、`Services/StudentService.cs`、`Data/Repositories/StudentRepository.cs`、`Models/Student.cs`。
- 数据库表：`Students`、`FamilyInfos`、`RewardRecords`、`PunishmentRecords`、`HealthRecords`。
- 依赖关系：StudentArchiveView -> StudentService -> StudentRepository -> DbContext。
- 约束：家庭/奖励/处分/体检列表展示和关联使用 `StudentNo`，不展示 `StudentId`。

## 学籍管理模块

- 功能范围：学籍注册、学籍变动、奖助学金、毕业信息列表、搜索、增删改查。
- 涉及文件：`Views/StudentStatus/*`、`Services/StudentStatusService.cs`、`Data/Repositories/StudentStatusRepository.cs`、`Models/StudentStatus.cs`。
- 数据库表：`StudentRegistrations`、`StatusChangeRecords`、`ScholarshipInfos`、`GraduationInfos`。
- 依赖关系：StudentStatusView/EditWindow -> StudentStatusRepository -> DbContext。
- 待完善：尚未统一进入 Service 层。

## 成绩管理模块

- 功能范围：成绩查询、录入、修改、删除、详情、CSV 导入导出、成绩等级计算。
- 涉及文件：`Views/Score/*`、`Services/ScoreService.cs`、`Data/Repositories/ScoreRepository.cs`、`Models/Score.cs`。
- 数据库表：`Scores`、`Courses`、`LookupOptions`。
- 依赖关系：ScoreView/ScoreEditWindow -> ScoreService -> ScoreRepository -> DbContext。

## 基础数据模块

- 功能范围：院系、专业、班级 CRUD，通用字典读取。
- 涉及文件：`Views/BasicData/*`、`Services/BasicDataService.cs`、`Data/Repositories/BasicDataRepository.cs`、`Models/BasicData.cs`。
- 数据库表：`Departments`、`Majors`、`Classes`、`LookupOptions`。
- 依赖关系：BasicDataView -> BasicDataService -> BasicDataRepository -> DbContext。

## 查询统计模块

- 功能范围：学生条件查询、统计摘要、CSV 导出。
- 涉及文件：`Views/Query/QueryView.xaml.cs`、`Services/StudentService.cs`、`Services/BasicDataService.cs`、`Common/CsvExportHelper.cs`。
- 数据库表：`Students`、`Departments`、`Classes`。
- 依赖关系：QueryView -> StudentService/BasicDataService -> Repository。

## 日志模块

- 功能范围：应用启动、数据库初始化、异常、登录、关键增删改、导入导出记录。
- 涉及文件：`Common/AppLogger.cs`、`App.xaml.cs`、各 Service 和导入导出页面。
- 输出：运行目录 `Logs/StudentsManagerSystem.log`。

## 导入导出模块

- 功能范围：学生基本信息 CSV、成绩 CSV、查询结果 CSV。
- 涉及文件：`Common/CsvExportHelper.cs`、`StudentArchiveView.xaml.cs`、`ScoreView.xaml.cs`、`QueryView.xaml.cs`。
- 数据库表：按导入目标使用 `Students` 或 `Scores`。
