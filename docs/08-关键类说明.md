# 08-关键类说明

## 应用入口

### `App`

- 路径：[`StudentsManagerSystem/App.xaml.cs`](../StudentsManagerSystem/App.xaml.cs#L13)
- 作用：应用启动、全局异常处理、数据库初始化、登录流程、主窗口显示。
- 主要方法：[`OnStartup`](../StudentsManagerSystem/App.xaml.cs#L18)。
- 对外能力：提供 [`CurrentUserRole`](../StudentsManagerSystem/App.xaml.cs#L14)、[`CurrentUserDisplayName`](../StudentsManagerSystem/App.xaml.cs#L16) 静态属性。
- 依赖：[`AppLogger`](../StudentsManagerSystem/Common/AppLogger.cs#L9)、[`DatabaseInitializer`](../StudentsManagerSystem/Data/DatabaseInitializer.cs#L14)、[`LoginWindow`](../StudentsManagerSystem/Views/Login/LoginWindow.xaml.cs#L9)、[`MainWindow`](../StudentsManagerSystem/MainWindow.xaml.cs#L19)。

### `MainWindow`

- 路径：[`StudentsManagerSystem/MainWindow.xaml.cs`](../StudentsManagerSystem/MainWindow.xaml.cs#L19)
- 作用：主界面导航、时间更新、基础权限控制。
- 主要方法：[`ApplyPermissions`](../StudentsManagerSystem/MainWindow.xaml.cs#L37)、[`MenuButton_Click`](../StudentsManagerSystem/MainWindow.xaml.cs#L66)。
- 依赖：[`MainViewModel`](../StudentsManagerSystem/ViewModels/MainViewModel.cs#L11)、各功能页。

## 数据核心

### `StudentsManagerDbContext`

- 路径：[`StudentsManagerSystem/Data/StudentsManagerDbContext.cs`](../StudentsManagerSystem/Data/StudentsManagerDbContext.cs#L9)
- 作用：EF Core 上下文和实体映射。
- 主要方法：[`OnConfiguring`](../StudentsManagerSystem/Data/StudentsManagerDbContext.cs#L49)、[`OnModelCreating`](../StudentsManagerSystem/Data/StudentsManagerDbContext.cs#L57)。
- 对外能力：提供所有业务实体的 `DbSet`。

### `DatabaseInitializer`

- 路径：[`StudentsManagerSystem/Data/DatabaseInitializer.cs`](../StudentsManagerSystem/Data/DatabaseInitializer.cs#L14)
- 作用：创建数据库、创建表、补齐轻量结构、写入种子数据。
- 主要方法：[`Initialize`](../StudentsManagerSystem/Data/DatabaseInitializer.cs#L16)、`SeedLookupOptions`、`SeedStudents`、`SeedScores`。
- 约束：唯一默认数据维护入口。

### `StudentsManagerDbContextFactory`

- 路径：[`StudentsManagerSystem/Data/StudentsManagerDbContextFactory.cs`](../StudentsManagerSystem/Data/StudentsManagerDbContextFactory.cs#L10)
- 作用：创建配置好 SQLite 连接的 DbContext。

## Service

### `StudentService`

- 路径：[`StudentsManagerSystem/Services/StudentService.cs`](../StudentsManagerSystem/Services/StudentService.cs#L8)
- 作用：学生查询、保存、导入合并、学生档案子表读取、校验、日志。
- 主要方法：[`GetAll`](../StudentsManagerSystem/Services/StudentService.cs#L13)、[`Search`](../StudentsManagerSystem/Services/StudentService.cs#L15)、`Add`、`Update`、`SaveImported`、`Validate`。
- 依赖：[`StudentRepository`](../StudentsManagerSystem/Data/Repositories/StudentRepository.cs#L10)、[`InputValidator`](../StudentsManagerSystem/Common/InputValidator.cs#L7)、[`AppLogger`](../StudentsManagerSystem/Common/AppLogger.cs#L9)。

### `ScoreService`

- 路径：[`StudentsManagerSystem/Services/ScoreService.cs`](../StudentsManagerSystem/Services/ScoreService.cs#L10)
- 作用：成绩查询、课程/学年/学期读取、成绩保存、导入合并、校验、日志。
- 主要方法：[`GetByAcademicYearSemester`](../StudentsManagerSystem/Services/ScoreService.cs#L19)、[`GetCourses`](../StudentsManagerSystem/Services/ScoreService.cs#L21)、[`SaveImported`](../StudentsManagerSystem/Services/ScoreService.cs#L67)。
- 依赖：[`ScoreRepository`](../StudentsManagerSystem/Data/Repositories/ScoreRepository.cs#L10)。

### `BasicDataService`

- 路径：[`StudentsManagerSystem/Services/BasicDataService.cs`](../StudentsManagerSystem/Services/BasicDataService.cs#L8)
- 作用：基础数据 CRUD、字典读取、校验、日志。
- 主要方法：[`GetDepartments`](../StudentsManagerSystem/Services/BasicDataService.cs#L12)、[`GetLookupValues`](../StudentsManagerSystem/Services/BasicDataService.cs#L26)、[`AddDepartment`](../StudentsManagerSystem/Services/BasicDataService.cs#L29)、`UpdateClass`。
- 依赖：[`BasicDataRepository`](../StudentsManagerSystem/Data/Repositories/BasicDataRepository.cs#L10)、[`InputValidator`](../StudentsManagerSystem/Common/InputValidator.cs#L7)。
