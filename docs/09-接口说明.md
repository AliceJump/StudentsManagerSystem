# 09-接口说明

## 说明

本项目不是 Web API 或 Service 项目，没有 HTTP 接口、Controller 路由或外部 RPC 接口。本文档中的“接口”指应用内部对外调用入口：Service 方法、Repository 方法、UI 事件入口和配置项。

## Service 接口

### StudentService

- `GetAll()`：获取全部学生。
- `Search(string keyword)`：按学号、姓名、院系、专业、班级搜索学生。
- `GetFamilyInfos()`、`GetRewardRecords()`、`GetPunishmentRecords()`、`GetHealthRecords()`：读取学生档案子表。
- `SaveFamilyInfo(FamilyInfo familyInfo)`、`SaveRewardRecord(RewardRecord rewardRecord)`、`SavePunishmentRecord(PunishmentRecord punishmentRecord)`、`SaveHealthRecord(HealthRecord healthRecord)`：保存学生档案子表记录，新增和修改共用。
- `DeleteFamilyInfo(int id)`、`DeleteRewardRecord(int id)`、`DeletePunishmentRecord(int id)`、`DeleteHealthRecord(int id)`：删除学生档案子表记录。
- `Add(Student student)`：新增学生并校验。
- `Update(Student student)`：更新学生并校验。
- `SaveImported(Student student)`：导入时按学号新增或更新。
- `Delete(int id)`：删除学生和关联数据。
- `Validate(Student student, int excludeId)`：学生格式和重复校验。

### ScoreService

- `GetAll()`：获取全部成绩。
- `GetByAcademicYearSemester(string academicYear, string semester)`：按学年学期查询。
- `GetCourses()`：获取课程。
- `GetAcademicYears()`：获取成绩中已有学年。
- `GetSemesters()`：从字典读取学期。
- `Add/Update/Delete`：成绩增删改。
- `SaveImported(Score score)`：导入时按学号、学年、学期、课程编号合并。

### BasicDataService

- `GetDepartments()`、`GetMajors()`、`GetClasses()`：获取基础数据。
- `GetDepartmentNames()`、`GetMajorNames()`、`GetClassNames()`：获取下拉显示名称。
- `GetLookupValues(string category)`：获取通用字典。
- `Add/Update/Delete Department/Major/Class`：基础数据增删改。

## Repository 接口

Repository 只面向内部 Service 或 View 调用，不作为外部接口。

- `StudentRepository`：学生与档案子表数据访问。
- `ScoreRepository`：成绩、课程、学年、学期数据访问。
- `BasicDataRepository`：院系、专业、班级、字典数据访问。
- `StudentStatusRepository`：学籍相关数据访问。
- `UsersRepository`：登录验证。

## UI 事件入口

- `MainWindow.MenuButton_Click`：主导航。
- `StudentArchiveView.btnAdd_Click`、`btnEdit_Click`、`btnDelete_Click`、`btnView_Click`、`btnImport_Click`、`btnExport_Click`：学生档案操作。
- `ScoreView.btnQuery_Click`、`btnImport_Click`、`btnExport_Click`：成绩操作。
- `BasicDataView.btnAdd_Click`、`btnEdit_Click`、`btnDelete_Click`：基础数据操作。
- `QueryView.btnQuery_Click`、`btnReset_Click`、`btnExport_Click`：查询统计操作。

## 配置项

配置文件：`StudentsManagerSystem/App.config`

- `DatabaseFile`：SQLite 数据库文件名。
- `AdminUsername`：初始化管理员用户名。
- `AdminPassword`：初始化管理员密码。
- `StudentsManagerSystemDb`：SQLite 连接字符串。

## 外部依赖接口

- 文件系统：CSV 导入导出、日志写入。
- SQLite 数据库文件：默认 `StudentsManagerSystem.db`。
- WPF 文件对话框：`OpenFileDialog`、`SaveFileDialog`。
