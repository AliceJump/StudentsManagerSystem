## 典型异常处理

- 启动初始化异常：[`App.OnStartup`](../StudentsManagerSystem/App.xaml.cs#L18) 捕获，写日志并弹窗后 `Shutdown(-1)`。
- UI 未处理异常：[`DispatcherUnhandledException`](../StudentsManagerSystem/App.xaml.cs#L22) 写日志并弹窗。
- 导入导出异常：页面 catch 后写日志并弹窗。
