## 根目录

- 用途：解决方案、项目说明、开发文档和脚本。
- 主要文件：`StudentsManagerSystem.sln`、`README.md`、`StudentManagementSummary.md`、`dotnet-install.sh`。
- 层级归属：项目管理和文档，不属于运行时业务层。

## docs

- 用途：长期维护文档。
- 主要职责：记录功能、架构、数据库、调用链、规范、维护手册。
- 层级归属：文档。

## StudentsManagerSystem

- 用途：WPF 应用主项目。
- 主要职责：承载 UI、模型、服务、数据访问和通用工具。
- 层级归属：混合应用项目。

## Common

- 用途：通用工具。
- 文件：[`RelayCommand`](../StudentsManagerSystem/Common/RelayCommand.cs)、[`InputValidator`](../StudentsManagerSystem/Common/InputValidator.cs#L7)、[`CsvExportHelper`](../StudentsManagerSystem/Common/CsvExportHelper.cs#L11)、[`AppLogger`](../StudentsManagerSystem/Common/AppLogger.cs#L9)、[`AcademicYearHelper`](../StudentsManagerSystem/Common/AcademicYearHelper.cs)
- 层级归属：Infrastructure / Shared。
- 关系：被 ViewModel、View、Service、App 调用。

## Data

- 用途：数据库上下文、配置、初始化。
- 文件：[`StudentsManagerDbContext.cs`](../StudentsManagerSystem/Data/StudentsManagerDbContext.cs#L9)、[`DatabaseInitializer.cs`](../StudentsManagerSystem/Data/DatabaseInitializer.cs#L14)、[`DatabaseConfiguration.cs`](../StudentsManagerSystem/Data/DatabaseConfiguration.cs#L11)、[`StudentsManagerDbContextFactory.cs`](../StudentsManagerSystem/Data/StudentsManagerDbContextFactory.cs#L10)。
- 层级归属：Data / Infrastructure。
- 关系：Repository 依赖 [`StudentsManagerDbContextFactory`](../StudentsManagerSystem/Data/StudentsManagerDbContextFactory.cs#L10)。

## Data/Repositories

- 用途：Repository 数据访问实现。
- 说明：当前 Repository 使用 SQLite + EF Core。
- 层级归属：Data。

## Data/Migrations

- 用途：EF Core 数据库结构迁移文件。
- 层级归属：Data。
- 关系：通过本地 `dotnet-ef` 工具生成和应用。

## Models

- 用途：实体模型。
- 层级归属：Domain。
- 关系：DbContext 映射这些实体，View 和 Service 使用这些实体传递数据。

## Services

- 用途：业务逻辑层。
- 层级归属：Application。
- 关系：UI 调用 Service（如 [`StudentService`](../StudentsManagerSystem/Services/StudentService.cs#L8)、[`ScoreService`](../StudentsManagerSystem/Services/ScoreService.cs#L10)、[`BasicDataService`](../StudentsManagerSystem/Services/BasicDataService.cs#L8)），Service 调用 Repository。

## Themes

- 用途：WPF 样式资源。
- 层级归属：UI。

## ViewModels

- 用途：部分 MVVM 支撑。
- 当前状态：[`MainViewModel`](../StudentsManagerSystem/ViewModels/MainViewModel.cs#L11) 实际使用；[`StudentArchiveViewModel`](../StudentsManagerSystem/ViewModels/StudentArchiveViewModel.cs) 仍是骨架。
- 层级归属：Application/UI Presentation。

## Views

- 用途：WPF 页面和窗口。
- 层级归属：UI。
- 关系：页面通过事件调用 Service 或 Repository。

### Views/Login

- 用途：登录窗口。
- 主要职责：用户名密码输入、登录校验提示。
- 层级归属：UI。
- 关系：调用 [`UsersRepository`](../StudentsManagerSystem/Data/Repositories/UsersRepository.cs#L10)。

### Views/StudentArchive

- 用途：学生档案模块。
- 主要职责：学生基本信息 CRUD、档案子表展示、CSV 导入导出。
- 层级归属：UI。
- 关系：调用 [`StudentService`](../StudentsManagerSystem/Services/StudentService.cs#L8)。

### Views/StudentStatus

- 用途：学生学籍模块。
- 主要职责：注册、变动、奖助、毕业维护。
- 层级归属：UI。
- 关系：当前直接调用 [`StudentStatusRepository`](../StudentsManagerSystem/Data/Repositories/StudentStatusRepository.cs#L10)。

### Views/Score

- 用途：学生成绩模块。
- 主要职责：成绩查询、录入、导入导出。
- 层级归属：UI。
- 关系：调用 [`ScoreService`](../StudentsManagerSystem/Services/ScoreService.cs#L10)。

### Views/BasicData

- 用途：基础数据模块。
- 主要职责：院系、专业、班级维护。
- 层级归属：UI。
- 关系：调用 [`BasicDataService`](../StudentsManagerSystem/Services/BasicDataService.cs#L8)。

### Views/Query

- 用途：查询统计模块。
- 主要职责：学生条件查询、统计摘要、导出。
- 层级归属：UI。
- 关系：调用 [`StudentService`](../StudentsManagerSystem/Services/StudentService.cs#L8) 和 [`BasicDataService`](../StudentsManagerSystem/Services/BasicDataService.cs#L8)。

## bin / obj

- 用途：构建输出和中间产物。
- 维护要求：不应手工修改，不应作为功能源码分析依据。
