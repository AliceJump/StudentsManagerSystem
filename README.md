# 学生管理系统

## 项目概述

本系统是一个基于 WPF（Windows Presentation Foundation）技术开发的学生管理系统，使用 C# 语言和 .NET 8.0 框架构建。系统采用现代化的界面设计，当前已接入 SQLite + EF Core 数据层、登录系统和部分导入导出能力。

## 当前工作情况

- 当前阶段：界面原型已完成，正在持续接入真实业务与数据层。
- 已完成内容：主界面导航、五大功能模块页面、SQLite + EF Core 数据层、登录系统、基础种子数据、部分 CRUD、CSV 导入导出、删除确认提示框。
- 暂未完全实现：全部模块的完整导入导出、图表统计、主题切换与更完整的 MVVM 页面重构。
- 项目属性：课程用项目

## 技术栈

- **框架**: .NET 8.0 + WPF
- **语言**: C#
- **开发工具**: Visual Studio Code / Visual Studio 2022
- **界面设计**: XAML
- **数据层**: SQLite + EF Core

## 系统架构

```

│   ├── StudentArchive/       # 学生档案管理

│   │   ├── StudentEditWindow.xaml
│   │   └── StudentDetailWindow.xaml
│   ├── StudentStatus/        # 学籍管理
│   │   └── StudentStatusView.xaml
│   ├── Score/                # 成绩管理
│   │   ├── ScoreView.xaml

```

## 主要功能模块

### 1. 主界面
- 左侧导航菜单，包含所有功能模块入口
- 右侧内容显示区域，动态加载各模块界面
- 实时显示系统时间
- 现代化的UI设计，使用蓝灰色调

### 2. 学生档案管理模块

包含五个子模块：

#### 2.1 学生基本信息
- 显示学生的基本资料（学号、姓名、性别、出生日期等）
- 支持新增、修改、删除、查看详情等操作
- 包含院系、专业、班级等组织信息

#### 2.2 家庭信息
- 管理学生的家庭成员信息
- 记录关系人姓名、关系、联系方式、工作单位等

#### 2.3 奖励记录
- 记录学生获得的各类奖励
- 包含奖励类型、等级、原因、颁发单位等信息

#### 2.4 处分记录
- 管理学生处分信息
- 支持处分撤销日期和状态管理

#### 2.5 体检信息
- 记录学生体检数据
- 包含身高、体重、血型、视力、健康状况等

### 3. 学生学籍管理模块

包含四个子模块：

#### 3.1 学籍注册
- 管理学生每学期的注册信息
- 记录注册日期、学年、学期、状态等

#### 3.2 学籍变动
- 记录学籍变动情况（转专业、休学、复学等）
- 包含变动原因、审批状态等信息

#### 3.3 奖助学金
- 管理学生奖助学金信息
- 记录金额、发放日期、状态等

#### 3.4 毕业信息
- 管理学生毕业相关信息
- 包含毕业证书号、学位证书号等

### 4. 学生成绩管理模块

- 按学年学期管理学生成绩
- 支持平时成绩、考试成绩、总评成绩录入
- 自动计算成绩等级
- 提供导入导出功能（预留接口）

### 5. 基础数据管理模块

包含三个子模块：

#### 5.1 院系管理
- 管理学校的院系信息
- 记录院系编号、名称、负责人、联系电话等

#### 5.2 专业管理
- 管理各院系的专业设置
- 包含学制、学位类型等信息

#### 5.3 班级管理
- 管理班级信息
- 记录班主任、学生人数等

### 6. 信息查询统计模块

- 支持多条件组合查询
- 提供学生信息查询
- 支持按院系、专业、班级统计
- 提供查询结果导出功能（预留接口）

## 运行方式

### 开发环境要求
- .NET 8.0 SDK
- Visual Studio 2022 或 Visual Studio Code
- Windows 10/11

### 编译运行
```bash
# 编译项目
dotnet build

# 运行项目
dotnet run --project StudentsManagerSystem
```

或在 Visual Studio 中直接按 F5 运行。

## 使用说明

1. **启动系统**：运行程序后进入主界面
2. **选择功能**：点击左侧菜单切换不同功能模块
3. **查看数据**：每个模块都展示了示例数据
4. **新增数据**：点击"新增"按钮，在弹出窗口中填写信息
5. **修改数据**：选中记录后点击"修改"按钮
6. **删除数据**：选中记录后点击"删除"按钮，会弹出确认对话框
7. **查看详情**：选中记录后点击"查看详情"查看完整信息
